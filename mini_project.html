AUTOMATIC NUMBER PLATE RECOGNITION APPLIED TO PARKING SECTION

Mini Project


ABSTRACT

Automatic Number Plate Recognition (ANPR) is a mass surveillance system that captures the image of vehicles and recognizes their license number. This paper presents the implementation of ANPR in various places like malls, tolls etc. and it also presents a recognition method in which the vehicle plate image is obtained by a digital cameras and the image is processed to get the number plate information.

The ANPR algorithm works on steps like extraction of numbers processing via filtering, segmentation, and character recognition. A data base is used to store the vehicle information. First the number plate is detected from an input frame, then the character from the plate are extracted and then OCR is applied to recognize the extracted character.

After all the process, a data base is used to store the recognized vehicle number along with entry time and further exit time is also noted. Then using the entry time and the exit time, the parking fare is calculated, and a bill is generated using the Thermal printer.

2.1 NUMBER PLATE DETECTION :


2.1.1 EXISTING METHODOLOGIES :

1: Image Binarization:

When compared to processing a grayscale image, black and white image processing requires less time. So there is a process called image binarization which converts a grayscale image into a black and white image. This technique selects a threshold to categorise certain pixels as black and certain pixels as white. However, how to select the right threshold value for a specific image is the key issue. Selecting the ideal threshold value can sometimes be exceedingly challenging or even impossible. To solve this issue, adaptive thresholding can be applied. A threshold can be manually chosen by the user or automatically chosen by an algorithm, which is referred to as automatic thresholding.

2: Hough Transform:

Hough Transform is a feature extraction technique that was initially employed for line detection. Later, it was later expanded to locate positions for arbitrary shapes like circles and ovals. D.H. Ballard generalised the original algorithm.

3: Blob Detection:

Blob detection is used to identify areas or points that stand out from their surroundings in terms of colour or brightness. The major goal of this method is to identify complementary regions that are not picked up by corner or edge detection techniques. Laplacian of Gaussian (LoG), Difference of Gaussians (DoG), Determinant of Hessian (DoH), maximally stable extremal regions, and Principle curvature based region detector are a few popular blob detectors.

4: Connected Component Analysis (CCA):

It is also known as blob extraction, a method to specifically name connected components' subsets based on a specific heuristic. It scans a binary image and labels each pixel in accordance with the connection conditions of the corresponding pixel, such as the current pixel's (8-connectivity) north, east, north, north west, and west Only the current pixel's north and west neighbours employ 4- connectivity. For automated picture analysis, the algorithm performs better and is highly helpful. Both character and plate segmentation can be done using this technique.

5: Mathematical morphology:

Set theory, lattice theory, topology, and random functions serve as the foundation for mathematical morphology. Although it is frequently utilised with digital images, it can also be applied to other spatial structures. It was initially created for the purpose of processing binary images before being expanded to process grayscale functions and images. It has fundamental operators including erosion, dilation, opening, and closing.


2.1.2 PROPOSED METHODOLOGY:

1: HAAR CASCADE ALGORITHM:

The Haar Cascade algorithm is a popular method for object detection in images, including the detection of license plates in images. The Haar Cascade algorithm is a kind of machine learning algorithm that trains a classifier using a set of positive and negative samples. Image which contain the required objects(license plate)are the positive samples, the image which doesn't contain the required object is the negative sample.

The algorithm uses the Haar wavelet approach to extract features from the positive and negative data during training. Haar wavelets are mathematical functions that is capable of spotting changes in contrast in an image. The system can find features that are likely to be present on licence plates, including the edges and corners of characters, by examining the variations in contrast at various points in a picture. Once trained,it can detect the number plate in new images.

The algorithm performs several scale and position scans of the image, computing a score at each point based on the existence of the features that were discovered during training. The algorithm assumes the area has a licence plate if the score is higher than a predetermined threshold. Overall, the Haar Cascade algorithm is a reliable and effective approach for detecting licence plates, and it has been applied to a variety of systems, including security systems and traffic monitoring.

2.2 CHARACTER RECOGNITION:

2.2.1 EXISTING METHODOLOGY:

1. Artificial Neural Network(ANN):

Artificial Neural Network (ANN), also referred to as neural network, is a mathematical concept that refers to a system of artificial neurons that are connected to one another. Character classification is done using an ANN model. It has three layers: an input layer for decision-making, a hidden layer to calculate more complex associations, and an output layer for the choice that is decided. An ANN was trained using the feed-forward back-propagation (BP) algorithm. The authors assert that their recognition rate is greater than 99%.

2. Template Matching:

Recognising fixed-sized characters is made possible through template matching. It can also be utilised for object detection, generally face detection and medical image processing. It is further divided into two sections: template-based matching and feature-based matching. The feature-based technique is effective when the template image includes prominent elements; otherwise, the template-based approach can be helpful.


2.2.2 PROPOSED METHODOLOGY:

1. OPTICAL CHARACTER RECOGNITION:

Optical Character Recognition (OCR) is a method used to identify characters or text in an image ( characters in license plate). To extract and recognise characters from images, OCR systems often combine image processing methods with machine learning algorithms.

· Character Recognition:

Apply the trained OCR algorithm to identify the characters from the new number plate image. The preprocessing of the image, character segmentation, feature extraction, and application of the OCR system to the extracted features are used to accomplish this. The character with the highest probability of being recognised is picked from a probability distribution over all potential characters that the OCR system outputs.

· Post-processing:

Refine the recognised characters using post-processing techniques, and get rid of any erroneous detections. Techniques like thresholding, morphological procedures, or language modelling can be used for this.

Overall, OCR offers a strong and adaptable framework for licence plate character identification. The OCR system's correctness is influenced by the calibre and volume of the training data, the architecture used for the OCR system, and the potency of the pre- and post-processing methods.


MERITS :

· Automation: OCR algorithms are capable of automating the reading of license plate numbers, removing the need for manual data entry and minimizing errors.

· Low computational demands: The Haar cascade approach is more suited for low-power devices and embedded systems since it uses less computing resources than other machine learning-based algorithms.

· Integration: These algorithms are perfect for usage in automated systems because they are simple to integrate with other technologies like cameras, sensors, and databases.


DEMERITS:

· Limitations on language and font: OCR algorithms may have trouble reading characters from other languages or fonts, especially if those characters are underrepresented in the training data.

· Limited adaptability: The Haar cascade method might not be able to accommodate modifications in license plate designs or variations in various regions or nations.
